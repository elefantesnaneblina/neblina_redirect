Teste de Universal Links / App Links

Esta é uma página de teste para verificar se os Universal Links (iOS) e App Links (Android) estão funcionando corretamente.

Se o app estiver configurado corretamente, clicar nos links abaixo deve abrir o app automaticamente.

Content Main Home

Para iOS: Tente pressionar e segurar o link para ver se aparece a opção "Abrir no app Neblina".

Para Android: Ao clicar, o app deve abrir automaticamente se a configuração estiver correta.

Se os links não funcionarem, verifique:

  • Se o app está instalado a partir do TestFlight/Google Play Testes
  • Se os arquivos de configuração estão corretos (apple-app-site-association e assetlinks.json)
  • Se as configurações no Info.plist e AndroidManifest.xml estão corretas
